# 05-3 分析仕様書：視聴回数別限界効果 × 期待効果分析

## 1. 分析目的

同じ予算で「既存医師への追加配信」と「新規医師への初回配信」のどちらが効果的かを定量化する。
具体的には、累積視聴回数ごとに「視聴が発生する確率（継続率）」と「視聴が生じた月の売上への効果（限界効果）」を推定し、その積として**期待効果**を算出する。

---

## 2. 分析対象集団の定義

### 2-1. フィルタリングの流れ

| ステップ | 条件 | ソースファイル |
|---|---|---|
| Step 1 | 施設内医師数 == 1 の施設に所属する医師のみ | `facility_attribute_修正.csv` |
| Step 2 | 所属施設数 == 1 の医師のみ | `doctor_attribute.csv` |
| Step 3 | RW医師リストに含まれる医師のみ（`INCLUDE_ONLY_RW=True` 時） | `rw_list.csv` |
| 最終確認 | 本院（fac_honin）単位で医師が1名のみのペアに限定 | `施設医師リスト.csv` |

Step 1・2 により「施設と医師が 1:1 対応」する集団に絞ることで、売上（施設単位）と視聴（医師単位）を一意に紐付ける。

### 2-2. 分析単位

- **医師 × 月次パネル**（全医師 × 全月の完全インデックス）
- 売上は施設コードで紐付け（施設と医師は 1:1 のため一意に対応）

---

## 3. パネルデータの変数定義

### 3-1. 主要変数

| 変数名 | 型 | 定義 |
|---|---|---|
| `doctor_id` | str | 医師ID |
| `month_id` | int | 月インデックス（0〜32） |
| `current_views` | int | **当月**の視聴回数（webinar + e-contents + Web講演会の合計） |
| `cumulative_views` | int | **当月含まず**の過去累積視聴回数 |
| `amount` | float | 施設の当月売上（品目コード 00001、単位：万円） |

### 3-2. `cumulative_views` の計算方法

```
cumulative_views[t] = sum(current_views[0], ..., current_views[t-1])
```

実装: `current_views` を医師ごとに `shift(1).fillna(0).cumsum()` で算出。
→ 当月の視聴は含まない（「この月の視聴が何回目か」を定義するため）

### 3-3. 視聴データの結合

- `デジタル視聴データ.csv`（webiner, e_contents）
- `活動データ.csv`（活動種別 == "Web講演会"）
の2ソースを結合し、品目コード・施設コード・医師IDで絞り込む。

## 4. 視聴回数ビニング（qcut 均等サンプル）

### 4-1. ビニングの目的

視聴回数別の限界効果をビン（区間）単位で推定する。各ビンのサンプル数が極端に少ないと推定精度が下がるため、`pd.qcut` で**サンプル数が均等になる境界**を自動決定する。

### 4-2. 手順

1. 視聴イベント行（`current_views > 0`）の `cumulative_views` を抽出
2. `pd.qcut(view_events_cum, q=N_BINS, duplicates='drop')` で N 等分
3. ビン境界を補正：
   - 左端を -0.5（`cumulative_views == 0` の初回視聴を含むため）
   - 右端をデータの最大値 + 0.5（最大値を含むため）
4. 各ビンのダミー変数を作成：
   `view_bin_k = 1` iff `cumulative_views ∈ (lo_edge, hi_edge]` AND `current_views > 0`

### 4-3. ビンラベルの読み方

| ラベル例 | 意味 |
|---|---|
| `累積0回→視聴（1回目）` | 過去0回視聴済みの医師が今月視聴 = 初回視聴 |
| `累積1-3回→視聴（2-4回目）` | 過去1〜3回視聴済みの医師が今月視聴 = 通算2〜4回目 |

## 5. 限界効果の推定（TWFE）

### 5-1. モデル式

$$
\text{amount}_{it} = \sum_{k=1}^{K} \beta_k \cdot \text{view\_bin\_k}_{it} + \alpha_i + \gamma_t + \varepsilon_{it}
$$

| 記号 | 意味 |
|---|---|
| $\text{amount}_{it}$ | 医師 $i$・月 $t$ の施設売上（万円） |
| $\text{view\_bin\_k}_{it}$ | ビン $k$ に該当する視聴ダミー |
| $\beta_k$ | ビン $k$ の視聴月における売上への**限界効果**（万円） |
| $\alpha_i$ | 医師固定効果（観察されない時間不変の医師特性を除去） |
| $\gamma_t$ | 月次固定効果（共通の時間トレンドを除去） |

### 5-2. 推定設定

| 項目 | 設定 |
|---|---|
| 推定ライブラリ | `linearmodels.PanelOLS` |
| 固定効果 | entity_effects=True（医師）, time_effects=True（月次） |
| 標準誤差 | クラスタード（医師クラスター、cluster_entity=True） |
| サンプル除外 | `amount == 0` の医師×月を除外（売上ゼロ月は活動なしとみなす） |

### 5-3. $\beta_k$ の解釈

- 「ビン k に該当する視聴があった月は、同じ医師の同じ季節（月次FE）と比べて売上が $\beta_k$ 万円高い」という相関関係
- **因果効果ではない**（視聴は内生変数：高意欲医師ほど視聴しかつ処方する傾向）

## 6. 継続率（視聴確率）の推定

### 6-1. 用語の整理

| 用語 | 定義 |
|---|---|
| **初回視聴確率**（initial_viewing_rate） | 未視聴の医師×月のうち、当月初回視聴が発生した割合 |
| **継続率**（continuation_rates[n]） | 累積視聴回数が n 回の医師×月において、翌月に視聴が発生する確率 |

### 6-2. 初回視聴確率の計算

```
分母: cumulative_views == 0 の医師×月数（過去一度も視聴していない月）
分子: そのうち current_views > 0 の行数
初回視聴確率 = 分子 / 分母
```

### 6-3. 継続率の計算

```python
dp_sorted["next_views"] = dp_sorted.groupby("doctor_id")["current_views"].shift(-1)

for n in range(0, max_cumulative + 1):
    cohort = 累積視聴回数が n 回の医師×月
    valid  = cohort の翌月視聴数（next_views）の非欠損値
    continuation_rates[n] = (valid > 0).mean()
```

**注意点：**
- `next_views` はシフトで付与するため、観測期間最終月の行は NaN になり除外される
- `n` の範囲はデータの最大累積回数まで動的に計算（固定値 15 だと最後のビンの確率がゼロになるバグが生じるため）

### 6-4. ビン別視聴確率

ビン $k$ の視聴確率は、そのビンの累積回数範囲 $[lo, hi]$ にわたる継続率の**平均**：

```
viewing_prob[bin_k] = mean(continuation_rates[n] for n in range(lo_cum, hi_cum+1))
```

ただしビン1（初回）のみ `initial_viewing_rate` を使用。

## 7. 期待効果の計算

### 7-1. 定義

$$
\text{期待効果}_k = P(\text{視聴}_k) \times \beta_k
$$

| 項目 | 意味 |
|---|---|
| $P(\text{視聴}_k)$ | ビン $k$ に該当する状態で視聴が発生する確率（継続率） |
| $\beta_k$ | ビン $k$ の限界効果（TWFE係数） |
| **期待効果** | 配信1回あたりの期待売上増（万円） |

### 7-2. 意思決定への応用

- 新規医師（累積0回）への配信期待効果 = `initial_viewing_rate × β_1`
- 既存医師（累積 n 回）への配信期待効果 = `continuation_rates[n] × β_k`（n が属するビン k）
- 「期待効果が新規医師を下回る最初のビン」を閾値として配信優先度を判断する

---

## 8. チャネル別期待効果の計算

### 8-1. 定義

チャネル $c$、ビン $k$ における期待効果：

$$
\text{期待効果}_{c,k} = P(\text{チャネル } c \text{ でビン } k \text{ に視聴}) \times \beta_k
$$

### 8-2. $P(\text{チャネル } c \text{ でビン } k \text{ に視聴})$ の計算

```
分母: ビン k に該当する医師×月数（doctor_panel でビン k の cumulative_views 条件を満たす行数）
分子: チャネル c の視聴のうち、累積回数がビン k の範囲に入る視聴行数の合計

P = 分子 / 分母
```

**解釈：** ビン k の状態にある医師×月のうち、チャネル c で視聴が発生する割合（視聴回数の重みつき）。

### 8-3. 各チャネルの合計

全チャネルの期待効果を合計すると、ビク別の全体期待効果と（概ね）一致する。
ただし分母の取り方（医師×月 vs 視聴行数ベース）により若干の差異が生じる可能性がある。

---

## 9. 内生性の注意事項

| 項目 | 説明 |
|---|---|
| **問題** | 視聴は医師の自発的行動（内生変数）。処方意欲の高い医師ほど視聴しかつ処方する |
| **TWFEで除去できるもの** | 医師固有の時間不変な特性（固定効果 $\alpha_i$）、全医師共通の時間トレンド（$\gamma_t$） |
| **除去できないもの** | 時変する交絡因子（例：特定の月に積極的になった医師が同時に視聴・処方する） |
| **結論** | $\beta_k$ は因果効果ではなく相関係数として解釈すること |
